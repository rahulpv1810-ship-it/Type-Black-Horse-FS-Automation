# Black Horse FS Automation Framework

An advanced AI prompt engineering system designed to autonomously convert raw Trial Balances into ICAI-compliant Schedule III financial statements.

## Overview
Currently iterating past v59+, this framework utilizes extended-context LLMs to perform complex, offline-first ledger mapping, forensic scrutiny, and statutory compliance checks for Indian corporate and non-corporate entities.

## Core Engines
* **Entity Detection:** Automatically adapts to Pvt Ltd, LLP, Partnership, Trust, and Proprietorship structures.
* **Substance-Over-Form Classification:** Intelligent ledger grouping mapping beyond basic nomenclature.
* **Forensic Anomaly Flagging:** Identifies high-risk transactions for audit scrutiny.
* **360° Pre-Filing Shield:** Integrates checks for Companies Act 2013, Form 3CD, and GST compliance.

## Current Status
The complete framework is currently utilized in private live audit engagements. Sample master prompts and system architecture guidelines will be published in this repository shortly.
